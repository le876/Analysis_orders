# 日收益率曲线（以首日总资产为本金 vs 真实净值）说明

## 1. 目的与展示
- 对比两套口径的策略表现：以“首日重估本金”构建的固定本金曲线 vs 真实净值曲线，并与基准指数净值、日收益率并排展示。
- 主图：净值曲线（首日=1），包含固定本金累积、真实净值、基准净值；副图：日收益率对比（策略两种口径 + 基准），零轴虚线标记。
- 目标：用统一本金视角评估下单质量与资金效率，同时保留真实净值口径便于与基准横向比较。

## 2. 数据与计算口径
- 输入来源：`lightweight_analysis.py` 生成的日级表，包含 `nav_capital_curve`、`nav_curve`、`daily_return_capital`、`daily_return_nav` 等字段；基准数据来自 `self.benchmark_data`（按交易日对齐，默认展示深证成指）。
- 本金口径：回放首日逐笔，求“当日最低所需本金”并乘安全系数 1.3；如重估失败回退为 `62,090,808` 元。
- 日收益率（固定本金）：`r_capital_t = 当日盈亏_t / 初始本金`（非复利），固定本金净值曲线 `= 1 + Σ r_capital_t`。
- 日收益率（真实净值）：`r_nav_t = (总资产_t - 总资产_{t-1}) / 总资产_{t-1}`（复利，首日设 0），真实净值曲线 `= Π (1 + r_nav_t)`。
- 基准对齐：以策略交易日为索引，将基准日收益/累计收益左连接并在图例中可切换。

## 3. 使用与产出
- 生成命令：`python src/lightweight_analysis.py`（自动写出页面与指标）。
- 输出路径：`reports/visualization_analysis/daily_returns_initial_capital_light.html`（仪表板用），`docs/daily_returns_initial_capital_light.html`（对外展示）。
- 指标卡：期末收益率（本金/净值）、夏普、最大回撤、日收益均值/波动，默认基准超额。
- 依赖：信用规则配置（授信规则）；基准数据可选。
