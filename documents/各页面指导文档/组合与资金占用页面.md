# 组合与资金占用页面说明

## 1. 覆盖页面与目的
- `capital_utilization_light.html`
- `portfolio_composition_light.html`
- `intraday_avg_holding_time_light.html`
- `amount_by_board_pie_light.html`
- `amount_by_industry_pie_light.html`
- `amount_by_market_cap_pie_light.html`
目的：展示组合持仓结构、资金占用与分行业/板块/市值维度分布，结合日内平均持仓时长评估风格与周转。

## 2. 数据与计算口径
- 输入：持仓/成交日级或分时汇总，含持仓市值、资金占用、成交金额、标的元数据（行业、板块、市值分组）。
- 主要指标：
  - 资金占用：最低所需本金或实时占用与当日收益的比值。
  - 组合构成：按市值/行业/板块的金额占比饼图。
  - 日内平均持仓时长：按标的或全局统计分钟数/天数分位。
- 元数据来源：本地静态映射或外部接口（已预处理为本地字段）。

## 3. 输出与位置
- 页面输出：`reports/visualization_analysis/` 与 `docs/` 对应文件名。
- 仪表板：归属“组合结构 / 资金效率”板块。
