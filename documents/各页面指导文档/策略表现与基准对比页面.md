# 策略表现与基准对比页面说明

## 1. 覆盖页面与目的
- `cumulative_returns_comparison_light.html`
- `strategy_vs_benchmark_light.html`
- `strategy_sharpe_nav.html`
- `recon_diagnosis.html`
目的：从累计收益、基准对比、夏普/净值以及回放一致性角度审视策略整体表现与数据质量。

## 2. 数据与计算口径
- 累计收益对比：策略与多基准的累计收益/超额，见 `累积收益对比页面修复记录.md` 的细节。
- 基准对比：策略净值与基准净值同轴展示，基准对齐按交易日。
- 夏普/净值：展示夏普、回撤等关键指标随时间或分组的表现。
- 回放诊断：校验回测与盯市、交易日志的一致性（如现金、持仓、费用），用于发现数据异常。
- 具体实现逻辑在 `src/lightweight_analysis.py` 对应生成函数。

## 3. 输出与位置
- 页面输出：`reports/visualization_analysis/` 与 `docs/` 对应文件名。
- 仪表板：策略表现/基准对比/诊断板块。
