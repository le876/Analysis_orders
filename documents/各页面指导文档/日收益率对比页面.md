# 日收益率对比页面说明

## 1. 覆盖页面与目的
- `daily_returns_comparison_light.html`
目的：对比多种下单质量或资金投放口径下的日收益率曲线，突出不同评估方法的差异；若需固定本金视角，请参考 `日收益率曲线（以首日总资产为本金 vs 真实净值）` 文档。

## 2. 数据与计算口径
- 输入：日度收益序列，可能含等权、金额加权、PnL/花出等多种口径。
- 计算：按日累积为净值曲线，同时展示日收益率；具体字段映射与口径定义见 `src/lightweight_analysis.py` 中对应生成逻辑。
- 基准：可选，与策略交易日对齐；若缺失则仅展示策略曲线。

## 3. 输出与位置
- 页面输出：`reports/visualization_analysis/daily_returns_comparison_light.html`（同步至 `docs/`）。
- 仪表板：收益对比/下单质量板块。

## 4. 修改历史与注意事项
- 2025-03-XX：创建说明文档。若调整收益口径或基准对齐方式，请同步更新此处。
