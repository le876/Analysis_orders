💼 页面目的

展示每日收盘时刻的资产负债快照，帮助评估策略收市时的资金占用、仓位结构以及调整成本，便于与收益表现结合诊断。

📈 图线含义

  • 现金余额（绿色）: 收盘后留存的现金头寸，体现策略在资金安全垫与融资需求上的取舍。
  • 多头持仓市值（蓝色）: 按收盘价估算的多头持仓规模，对应收盘时的多头敞口。
  • 空头持仓市值（红色）: 收盘时的空头持仓绝对金额，反映对冲或做空力度。
  • 每日交易费用（橙色，右轴）: 当日发生的全部手续费、滑点等成本，用于衡量仓位调整的代价。

🔍 使用建议

  • 收盘仓位节奏: 结合多头/空头曲线，判断策略是否在重要日期显著增减仓。
  • 资金安全垫: 关注现金曲线的低点与波动，评估是否存在资金吃紧或闲置。
  • 杠杆与对冲: 负现金或显著空头敞口提示杠杆使用程度，需要核对保证金占用。
  • 调整成本: 手续费与仓位变动同时放大时，应进一步分析是否产生额外的交易损耗。

⚙️ 数据生成流程

  1. 读取 mtm_analysis_results/daily_nav_revised.csv，提取收盘现金、多头市值、空头市值及总资产字段并统一为数值型。
  2. 使用正确的初始本金 62,090,808 元（基于首日最低所需本金 47,762,160 × 1.3），重新计算每日现金余额和总资产。
  3. 从订单明细中按日聚合手续费，合并至盯市数据，缺失日期填补为 0。
  4. 按时间排序后可选抽样（默认保留≤200个点），以在图表中展示完整的收盘序列。
