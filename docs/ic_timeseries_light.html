📋 数据处理详细过程

  1. 数据源: orders.parquet文件中的 pred 和 real 字段
  2. 预处理: 移除 pred 和 real 字段的缺失值
  3. 股票-日聚合: 按 (Code, 日期) 分组，对 pred 和 real 取平均值
  4. 前瞻收益计算: 对按股票分组后的 real 字段向后移动一期，得到下一交易日的真实收益用于T+1评估
  5. 横截面IC计算: 每个交易日，计算当日 pred 与下一日 real 的皮尔逊相关系数
  6. 数据采样: 若超过300个交易日，按步长采样保留250个点用于展示
  7. 移动平均: 计算10日滑动平均，最小窗口为1
  8. 图表生成: 使用Plotly生成时间序列图，包含原始IC和移动平均线

关键字段说明:

  • pred: 模型预测值，数值越高表示预期收益越好
  • real: 实际收益标签，由数据准备脚本基于未来价格变动计算
  • IC: Information Coefficient，衡量预测值与实际收益的相关性
