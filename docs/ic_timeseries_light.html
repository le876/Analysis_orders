📋 数据处理详细过程

  1. 数据源: 来源于 orders.parquet 中的 pred 与 real 字段
  2. 标签说明: 当前分析暂以 T+10 分钟收益作为临时标签参与 IC 计算，后续如标签口径调整将同步更新。
  3. 预处理: 剔除 pred、real 的缺失值，并统一时间戳格式
  4. 股票-日聚合: 按 (Code, 日期) 分组求均值，得到当日同一股票的代表性预测值与真实收益
  5. 横截面IC计算: 针对每个交易日，直接对该日内所有股票的 pred 与 real 计算皮尔逊相关系数
  6. 数据采样: 当交易日超过300天时，以固定步长采样，控制展示点位至约250个
  7. 移动平均: 对日度IC执行10日滑动平均，以观察平滑趋势
  8. 图表生成: 使用 Plotly 渲染日度IC、10日均线，以及极端分组（G1/G10）的追踪曲线
