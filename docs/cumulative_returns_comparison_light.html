累积收益对比（三种方法）

📊 计算方法说明

本页展示三种日收益序列的复利累积结果（即 $\prod(1 + r_i) - 1$），每条曲线的数据来源与计算方法如下：

方法1：等权收益（期末 45.88%）
  • 数据来源：从 data/orders.parquet 中按 Code 和 Timestamp 先后进行买卖配对（FIFO原则）
  • 单笔收益率：$r_{\text{单笔}} = \frac{\text{卖出}_{\text{tradeAmount}} - \text{买入}_{\text{tradeAmount}} - \text{总}_{\text{fee}}}{\text{买入}_{\text{tradeAmount}}}$
  • 日收益聚合：按平仓日期，对当日完成的所有交易对的收益率取简单平均
  • 优点：反映选股与下单方向的纯技术表现，不受资金配置影响
  • 局限：仅包含已平仓交易，未平仓浮动盈亏不计入
方法2：金额加权收益（期末 55.22%）
  • 数据来源：与等权收益相同的配对交易数据
  • 日收益聚合：按开仓时 tradeAmount 作为权重，计算加权平均日收益率
  • 优点：体现资金配置效果，大额交易对收益的影响更大
  • 局限：同样仅包含已平仓交易
方法3：PnL(花出的钱)（期末 0.21%）
  • 数据来源：从盯市文件读取每日总资产，计算总资产差分得到当日盈亏
  • 分母定义：当日 direction 为 B 的所有 tradeAmount 之和（当日买入总额）
  • 日收益率：$r_t = \frac{\text{当日盈亏}_t}{\text{当日买入总额}_t}$（仅在当日买入总额 > 0 时定义）
  • 累计口径：累计资金效率 = $\frac{\sum \text{当日盈亏}}{\sum \text{当日买入总额}}$
  • 优点：包含全部持仓（已平仓+未平仓），真实反映现金投入回报
  • 适用场景：资金利用效率监控、真实盈利能力评估

📌 关键假设

  • 累积前对日收益序列进行裁剪至 $[-90\%, 90\%]$ 区间，以抑制极端噪声对复利计算的影响（仅影响展示，不改变原始数据）
  • 前两种方法基于配对交易，仅统计已完成的买卖对；第三种方法基于盯市总资产，包含全部持仓
  • 三种方法的差异反映了"交易完成度"、"资金配置效率"与"真实现金效率"的不同视角

📖 解读建议

  • 方法接近：当三条曲线趋势一致时，说明策略收益结构稳健，已平仓与未平仓收益方向一致
  • 方法分离：若PnL(花出的钱)显著偏离，可能是未平仓浮动盈亏较大，或当日资金投入与平仓节奏不匹配
  • 建议组合使用：等权看选股能力，金额加权看配置效率，PnL(花出的钱)看真实回报
