口径与结算说明

1) 指标定义
  • 每日最低所需本金：对每个交易日，按 tradeTimestamp 或 Timestamp 先后逐笔回放资金、持仓与价格，在任意时点计算 所需权益 = $\max(0, -\text{可用现金}) + \text{空头保证金}$，其中空头保证金 = 空头市值 × 保证金比例，并取 当日峰值 作为该日的最低所需本金。
  • 日度资金占用收益率：$\text{日度收益率}_t = \dfrac{\text{当日盈亏}_t}{\text{当日最低所需本金}_t}$；其中当日盈亏 = 总资产$_t$ - 总资产$_{t-1}$，总资产基于正确的初始资金重算；当日分母≤0或缺失时，结果记为NaN。
  • 累计资金占用收益率：$\text{累计收益率}_t = \sum\limits_{i \le t} \text{日度收益率}_i$（算术累计，非复利）。
2) 逐日结算流程（摘要）
  1. 按时间排序处理当日每笔成交；direction 为 B（买入）：减少可用现金；卖出平多：按规则(如T+0)回补现金；开空：按规则计入/不计入现金，仅计提 fee。
  2. 每处理一笔，更新持仓与最新 price，并据此得到当前空头市值与保证金（空头市值 × 保证金比例）。
  3. 根据费用计提规则（逐笔/日终），计算可用现金，并据此得到 所需权益 = $\max(0, -\text{可用现金}) + \text{空头保证金}$，记录当日最大值。
  4. 结合重算后的总资产序列得到当日盈亏，据此计算日度与累计资金占用收益率。
3) 本页使用的授信/清算规则
  • 空头保证金比例：50%
  • 卖出平多现金可用性（T+0）：是
  • 开空所得计入可用现金：否
  • 费用计提口径：逐笔(实时)
  • 初始资金安全系数（回推用）：1.30
  • 借券费/融券利息计入：未配置
